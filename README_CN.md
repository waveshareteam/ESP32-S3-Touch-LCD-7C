# Waveshare ESP32-S3-Touch-LCD-7C 产品工程示例程序

[English](README.md)

这款产品为 **ESP32-S3-Touch-LCD-7C**，基于 ESP32-S3 的 7 英寸 RGB LCD 电容触控开发板，可用于 GUI（如 LVGL）开发与外设功能验证。
